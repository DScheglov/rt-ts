# ts-cast: Basics (example)

## Install and Start

```sh
git clone https://github.com/DScheglov/ts-cast.git
cd ts-cast/examples/either
npm install
npm start
```

**Output**

```shell
Book {
  title: 'My First Book',
  annotation: undefined,
  year: 2021,
  authors: [ Person { name: 'John Smith', email: 'john@smith.com' } ],
  coords: [ 1.2, 23.32 ]
}
```
